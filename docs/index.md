# Welcome to the _breseq_ user manual!

_breseq_ is a computational pipeline for finding mutations in haploid microbial
genomes by comparing short-read re-sequencing data to an already sequenced
reference genome. See the [Introduction](introduction.md) for a full description
of what _breseq_ does and when to use it.

New to _breseq_? Start with [Installation](installation.md) and then walk through
the [Test Drive](test-drive.md) example.

Looking to contribute to _breseq_ itself? See
[CONTRIBUTING.md](https://github.com/barricklab/breseq/blob/master/CONTRIBUTING.md)
on GitHub.
